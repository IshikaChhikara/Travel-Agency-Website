# 🌍 Travel Agency Website

A responsive travel booking website built using HTML, CSS, and JavaScript.

## ✨ Features
- Booking system (half payment)
- Time-based refund policy
- Dark mode toggle
- Wishlist system
- Clean and responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
